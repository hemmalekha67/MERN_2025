R&D QUESTIONS

  1. Why Git Shows "Unstage" Instead of "Commit" After Adding a File?

    Files are already staged and ready for a commit, so Git suggests "unstage" instead.

  2. Difference Before and After the First Push

    Before the first push, no link exists between the local and remote repository. After the first push, a link is established.

  3. Rendering Title and H1 in Improper HTML Structure

    The browser automatically assigns the <title> to the head and the <h1> to the body based on default HTML parsing rules.

  4. Difference Between href and src

    href (in anchor tags) redirects to external links, while src (in img tags) embeds content within the webpage.

  5. Why Are There "..." in if...else and do...while?

    The "..." indicates omitted code or logic, implying additional statements should be placed there.

  6. Difference Between Spread and Rest Operators

    The spread operator (...) expands arrays or objects. The rest operator (...) collects elements into an array and is used in function parameters.

  7. For-In Loop with Dot Operator Outputs Undefined

    The dot operator accesses a property literally named "key" instead of dynamically accessing the property using the variable's value.
