8. If the following is typed in an HTML file without its basic structure, what will appear in the inspect tab?

  1. <h1>Hello world!</h1>
  2. <title>sece2025</title>
    <h1>Hello world!</h1>
  3. <h1>Hello world!</h1>
    <title>sece2025</title>

Answer:
i. <html>
<head></head>
<body><h1>Hello world!</h1></body>
</html>

ii. <html>
<head><title>sece2025</title></head>
<body><h1>Hello world!</h1></body>
</html>

iii. <html>
<head></head>
<body><h1>Hello world!</h1><title>sece2025</title></body>
</html>
Reason: The <title> tag appears in the <body> when inspecting because the browser encounters visible content like <h1> first, causing it to start rendering the <body> implicitly before processing the <title>.

R&D QUESTIONS

1. Why is Git not showing the git commit command while adding a file and instead showing a command for unstage?

Git is not showing the git commit command because files are already staged and ready for a commit.

2. What is the difference before the first push and after the first push?

Before the first push, there is no link between the local and remote repository. After the first push, there is a link established between the local and remote repository.

3. How are the <title> tag and <h1> tag fetched when rendering an HTML file with no structure?

The browser automatically places the <title> tag into the head and the <h1> tag into the body based on HTML parsing rules and defaults.

4. What is the difference between href and src, and why are they used in img and anchor tags?

href is used with the anchor tag to redirect to an external link, while src is used with img tags to embed content within a webpage.

5. Why are there three dots in if...else and do...while?

The three dots (...) in if...else and do...while in documentation or examples are placeholders used to indicate omitted code or logic. They imply additional statements or blocks should be added in those places.

6. What is the difference between the spread and rest operator?

The spread operator (...) expands an array or object, while the rest operator (...) collects multiple elements into an array or parameters and is used as a function parameter.

7. In a for-in loop, why is the output undefined when accessing an object property using the dot operator?

The output is undefined because obj.key looks for a property literally named "key" in the object instead of dynamically accessing the property using the key variable.

8. Why should components start with a capital letter?

Components should start with a capital letter because React uses capitalized names to differentiate components from standard HTML tags.

9. Why is the default keyword used while exporting a component?

The default keyword is not mandatory; it specifies a default export, allowing the component to be imported without curly braces, unlike named exports which require them.
